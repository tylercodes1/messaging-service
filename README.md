# messaging-service API
This is a messaging service API for my future social media-esque website

## Technologies
* Spring
  * Spring Boot
    * Spring Starter Data JPA
    * Spring Starter Web
    * Spring DevTools
    * AspectJ (AOP)
* slf4j (Logging)
* Lombok
